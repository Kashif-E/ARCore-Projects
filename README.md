# ARCore-Projects
This repo uses sceneform 1.17.1 to demonstrate the usage of different features of arcore such as,

* Augmented Faces
* Augmented images
* AR View
* 3d View 
* Transformable nodes
* nodes 
* anchor nodes
* hit test 
 and many more..
